# Introduction

Les diagrammes de séquence permettent de représenter les interactions entre les différents composants du système au cours d’un scénario précis. Ils décrivent l’enchaînement des échanges entre les acteurs, le frontend, le backend et la base de données.

Dans le cadre du projet NexTeam, ces diagrammes ont été réalisés afin de modéliser les principaux processus fonctionnels.
