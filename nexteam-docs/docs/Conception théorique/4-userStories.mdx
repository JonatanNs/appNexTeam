# User stories

Auteur : Jonatan NSUALU

Date : Mai 2026

# Présentation des user stories

Les user stories décrivent les besoins fonctionnels des différents acteurs de la plateforme (Employé, Manager, RH et Administrateur). Elles permettent de définir les fonctionnalités attendues du système du point de vue utilisateur.

---

## Employé

En tant qu’employé,
je veux me connecter afin d’accéder à la plateforme.

En tant qu’employé,
je veux consulter les actualités afin de rester informé des informations de l’entreprise.

En tant qu’employé,
je veux envoyer un message afin de communiquer avec les autres membres de l’entreprise.

En tant qu’employé,
je veux supprimer un message afin qu’il ne soit plus disponible.

En tant qu’employé,
je veux mettre à jour l’état d’une tâche afin de montrer l’avancement de mon travail.
