# Dépendances Spring Boot — ConnecTeam

## Dépendances principales

| Dépendance | Utilité |
| --- | --- |
| Spring Web | Création API REST |
| Spring Security | Authentification et sécurisation API |
| Spring Data JPA | Communication avec la base de données |
| PostgreSQL Driver | Connexion PostgreSQL |
| Validation | Validation des données (`@Valid`) |
| Lombok | Réduction du boilerplate Java |
| WebSocket | Communication temps réel |
| SpringDoc OpenAPI | Documentation Swagger/OpenAPI |
| Java Mail Sender | Emails et notifications |
